# unit

### getName
```lua
unit:getName()
    -> name: string
```

### set
```lua
unit:set(attributeName: string, attributeValue: number)
```

### get
```lua
unit:get(attributeName: string)
    -> value: number
```

### add
```lua
unit:add(attributeName: string, attributeValue: number)
```

### isAlive
```lua
unit:isAlive()
    -> boolean
```

### isHero
```lua
unit:isHero()
    -> boolean
```

### kill
```lua
unit:kill([target: unit])
```

### remove
```lua
unit:remove()
```

### getPoint
```lua
unit:getPoint()
    -> point
```

### setPoint
```lua
unit:setPoint(point)
```

### getOwner
```lua
unit:getOwner()
```

### particle
